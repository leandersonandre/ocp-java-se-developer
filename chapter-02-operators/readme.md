Operador ^ or exclusive (XOR)
```
# differents are equal
var a = true;
var b = false
booleand c = a ^ b; # true
# equals are false
var a = true;
var b = true
booleand c = a ^ b; # false
```

Bitwise Complement Operator (~ tilde)
```
var i = 10;
var j = ~i; # -11
# Multiply by -1 and plus -1.
# (10 * -1 ) + (- 1) = -11

var i = -10;
var j = ~i; # 9
# Multiply by -1 and plus -1.
# (-10 * -1 ) + (- 1) = 9
```
